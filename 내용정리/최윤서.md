#### 1. 기본 문법
###### (1)
public class ClassName {  
  public static void main(String[] args) {  
    //실행코드  
    }  
}  

###### (2) 출력
- System.out.println(); //줄바꿈O  
- System.out.print(); //줄바꿈X  
- System.out.printf(); //서식(format) 지정  

###### (3) 주석
- // : 한 줄 주석  
- /* */ : 여러 줄 주석

### 2. 변수와 자료형
###### (1) 변수명 규칙
- 문자, 숫자, _, $만 사용 가능
- 숫자로 시작하면 안 됨
- 대소문자 구분함 (age ≠ Age)
- 예약어(예: int, class)는 사용 불가

###### (2) 자료형
- int num = 10; //정소
- double d = 3.14; //실수
- char C = 'A'; //문자
- boolean flag = true; //논리

###### (3) 형변환
- int i = (int) 3.7; //명시적 형변환
- double d2 = 5; //자동 형변환

###### (4) 문자열 비교
equals()로 비교
== 는 주소 비교

### 3. 연산자
###### (1) 우선순위
산술 -> 비교 -> 논리

###### (2) 연산자
- 산술 : +, -, *, /, %
- 증감 : ++, --
- 비교 : ==, !=, >, <, >=, <=
- 논리 : &&, ||, !
- 삼향 : int max = (a>b) ? a:b
